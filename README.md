***HOW TO NAVIGATE***

1. Use **W** **A** **S** **D** to walk around the 3d space
2. Use **F** to interact with objects that interacts with the user
3. Use **CLICK** to throw balls
4. Use **E** to acces the menu
5. Use **Q** to lock the cursor
